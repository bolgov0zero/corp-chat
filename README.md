## Структура проекта

```
Electron-Messenger/
├── server/                   ← серверная часть (Node.js)
│   ├── src/
│   │   ├── index.js          точка входа
│   │   ├── db.js             база данных SQLite

│   │   ├── routes/           REST API + admin
│   │   └── public/           веб-клиент и PWA

    ├── main.js
    ├── src/
    └── package.json
```

## Установка сервера на Linux

### 1. Клонировать репозиторий

```bash
git clone https://github.com/bolgov0zero/Electron-Messenger.git /opt/electron-messenger
cd /opt/electron-messenger/server
```

### 2. Запустить установку

```bash
sudo bash install.sh
```

Скрипт автоматически:
- Установит Node.js 20, если его нет
- Установит зависимости (`npm install`)
- Создаст и запустит systemd-сервис `electron`
- Зарегистрирует команду `electron` для управления сервером

**Поддерживаемые дистрибутивы:** Ubuntu / Debian, CentOS / RHEL / Rocky Linux, Arch Linux

После установки сервер будет доступен по адресу:
```
http://<IP-сервера>:3000
```

## Управление сервером

После установки используй команду `electron` прямо из консоли:

```bash
electron
```

Откроется интерактивная панель управления:

```
╔══════════════════════════════════════════════╗
║         ⚡  Electron  Server                 ║
╚══════════════════════════════════════════════╝

●  Работает   14:22:01

    Версия        1.2.070
    Аптайм        2д 4ч 17м
    Порт          3000

    Пользователи  12
    Сообщений     4821
    Чатов         7       1.2M

  [1] Остановить    [2] Перезапустить
  [3] Обновить      [4] Проверить обновления
  [5] Логи          [0] Выйти
```

Опция **[3] Обновить** — скачивает последнюю версию с GitHub и перезапускает сервер без потери данных.

Также доступны стандартные команды systemd:

```bash
systemctl status electron
systemctl restart electron
journalctl -u electron -f
```

## Веб-панель администратора

```
http://<IP-сервера>:3000/admin
```

**Логин по умолчанию:** `admin` / `admin`

> Смени пароль сразу после первого входа: раздел «Пользователи» → «Пароль» напротив admin.

| Раздел | Возможности |
|--------|-------------|
| Главная | Статистика: пользователи, чаты, сообщения |
| Пользователи | Создать, удалить, сменить пароль |
| Чаты | Просмотр, удаление, очистка истории |
| Настройки | GitHub-токен, время редактирования, обновление клиента |

## Клиент Electron

Десктопный клиент собирается через GitHub Actions при создании тега `c*`.

Готовые сборки доступны на странице [Releases](https://github.com/bolgov0zero/Electron-Messenger/releases).

### Установка на macOS (Apple Silicon)

1. Скачай `.dmg` из последнего релиза
2. Открой, перетащи **Electron** в папку **Applications**
3. При первом запуске: правая кнопка мыши → **Открыть** → **Открыть**

Или через терминал, если macOS заблокировал:
```bash
xattr -d com.apple.quarantine /Applications/Electron.app
```

### Первый вход

1. Запусти приложение
2. В поле **«Адрес сервера»** введи IP и порт:
   ```
   192.168.1.10:3000
   ```
3. Введи логин и пароль → **Войти**

## Переменные окружения

Задаются в `/etc/systemd/system/electron.service` или перед запуском:

| Переменная | По умолчанию | Описание |
|-----------|-------------|----------|
| `PORT` | `3000` | Порт сервера |
| `JWT_SECRET` | генерируется при установке | Секрет для JWT-токенов |
| `DB_PATH` | `../chat_db/chat.db` | Путь к базе данных SQLite |

После изменения переменных в сервисе:

```bash
systemctl daemon-reload
systemctl restart electron
```

### Firewall

Если на сервере настроен firewall, открой порт:

```bash
# Ubuntu / Debian
sudo ufw allow 3000/tcp

# CentOS / RHEL
sudo firewall-cmd --permanent --add-port=3000/tcp
sudo firewall-cmd --reload
```
